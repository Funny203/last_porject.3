# Schedule Bot
## Подготовление
pip install telebot
### 1. Получение токена
1. Найдите в Telegram [@BotFather](https://t.me/botfather)
2. Отправьте команду `/newbot`
3. Придумайте название бота (прим: "Школьное расписание")
4. Придумайте username бота (прим: "myschool_bot")
5. Скопируйте полученный токен (выглядит так: `7234567890:AAHdqT.....`)

### 2. Настройка
1. Вставьте токен в файл `config.py: token = "7234567890:AAHdqT....."`
