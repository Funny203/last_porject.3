# Telegram-бот для расписания онлайн-школы

## Что делает бот?
Бот выдаёт случайное расписание уроков от 4 до 6 предметов

## Как получить токен?
1. Найди в Telegram **@BotFather**
2. Отправь команду `/newbot`
3. Придумай **название** (например, "Школьное расписание")
4. Придумай **username** (обязательно заканчивается на `bot`, например `myschool_bot`)
5. Скопируй полученный токен (выглядит так: `7234567890:AAHdqTcvCH1vGWJxfSeofSAs0K5PALDsaw`)

## Как начать?
1. Вставь токен в `config.py`
2. Установи библиотеку: `pip install pytelegrambotapi`
3. Запусти: `python main.py`
